==该项目是一个前端项目==
该项目里面没有什么页面，主要是用于测试前后端分离项目时登陆相关逻辑。
主要有elementui，router，axios；
配置有axios的请求拦截器，相应拦截器，调本地后端的跨域处理，动态路由，router跳转时是否登陆

#该项目和另一个security后端项目结合使用，能够更好的熟悉spring security
#该项目的部署和后端的部署文件需修改注意事项 放在 deployment.md 文件中

## Build Setup

npm run unit

# run all tests
npm test
